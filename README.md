# Nexus - Business Management Platform

A modern, full-stack business management platform built with React, Vite, and Tailwind CSS. Nexus helps teams collaborate, manage projects, track payments, and grow their business all in one place.

**Live Demo**: [https://nexus-opal-seven-38.vercel.app](https://nexus-opal-seven-38.vercel.app)

---

## ✨ Features

- **Dashboard**: Real-time overview of business metrics
- **Project Management**: Kanban board to track tasks and projects
- **Team Collaboration**: Chat, video calls, and document sharing
- **Calendar**: Schedule meetings and manage deadlines
- **Payment Tracking**: Monitor invoices and payments
- **Document Management**: Upload, share and manage business docs
- **Responsive Design**: Works perfectly on desktop and mobile
- **Dark/Light Mode**: Toggle between themes

## 🛠️ Tech Stack

- **Frontend**: React 18, Vite, Tailwind CSS
- **UI Components**: shadcn/ui, Radix UI
- **Icons**: Lucide React
- **Routing**: React Router DOM
- **State Management**: React Context + Hooks
- **Deployment**: Vercel

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

```bash
# Clone the repository
git clone https://github.com/fzounr5-source/Nexus.git

# Go to project directory
